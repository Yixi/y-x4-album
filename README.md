# Y-X4-Album

把你的阅星瞳 X4 变成一台优雅的电子相册。

> **WIP** — 项目处于早期开发阶段，核心功能可用，但仍在快速迭代中。欢迎试用和反馈。

---

## 它是什么

Y-X4-Album 是一款为 [阅星瞳 X4](https://www.xteink.com/) E-Ink 设备打造的**开源电子相册固件**。

把照片丢进 TF 卡，刷入固件，你就拥有了一个超长续航、阳光下清晰可见、没有蓝光的随身相册——或者一个不需要充电线的桌面相框。

## 为什么用 E-Ink 看照片

- **纸质般的观感** — 电子墨水屏让照片看起来像印在纸上，黑白灰度有独特的艺术质感
- **断电也能显示** — 关机后屏幕保持最后一张图片，当作相框零功耗
- **阳光下更清晰** — 户外、窗边，越亮越好看，和手机屏幕完全相反
- **真正的护眼** — 无背光、无频闪，适合放在床头或书桌
- **续航以周计** — 不用天天充电

## 功能

- [x] **图库浏览** — 缩略图网格，一页 15 张，快速翻阅
- [x] **全屏查看** — 4 级灰度抖动渲染，最大化图片细节
- [x] **幻灯片** — 自动播放，顺序/随机，可设间隔（5 秒 ~ 5 分钟）
- [x] **格式支持** — JPEG、PNG、BMP，自动识别
- [x] **递归扫描** — 自动发现 TF 卡所有子目录中的图片
- [x] **文件浏览器** — 手动进入指定文件夹浏览
- [x] **设置** — 显示模式、排序方式、幻灯片参数、屏幕方向、语言等
- [x] **中英双语** — 界面支持中文和英文
- [x] **电源管理** — 自动休眠、按键唤醒、低电量保护

## 硬件要求

| 项目 | 要求 |
|------|------|
| 设备 | 阅星瞳 X4 |
| TF 卡 | FAT32 或 exFAT 格式，存入图片即可 |
| 数据线 | USB-C（需支持数据传输，非纯充电线） |

## 快速开始

### 刷入固件

1. 唤醒 X4，用 USB-C 连接电脑
2. 克隆代码并编译烧录：

```bash
git clone https://github.com/yixi/y-x4-album.git
cd y-x4-album
pio run --target upload
```

> 需要安装 [PlatformIO](https://platformio.org/)（CLI 或 VS Code 插件）

### 准备照片

把图片放入 TF 卡（支持子文件夹），插入 X4，开机即可浏览。

建议：
- JPEG 格式体验最佳（文件小、解码快）
- 分辨率接近 800×480 效果最好，大图会自动缩放
- 黑白或高对比度照片在 E-Ink 上特别好看

## 操作方式

X4 通过物理按钮操作：

| 界面 | 上/下 | 左/右 | 确认 | 返回 |
|------|-------|-------|------|------|
| 图库 | 切换行 | 移动焦点 | 打开图片 | 文件浏览器 |
| 全屏 | 上/下一张 | 上/下一张 | 显示信息 | 返回图库 |
| 幻灯片 | — | — | 暂停/继续 | 退出 |
| 设置 | 上下选择 | 调整值 | 确认修改 | 返回 |

## 技术栈

- **MCU**: ESP32-C3（RISC-V, 160MHz, 380KB RAM）
- **显示**: 800×480 E-Ink，4 级灰度抖动
- **框架**: PlatformIO + Arduino + C++20
- **SDK**: 基于 [crosspoint-reader](https://github.com/crosspoint-reader/crosspoint-reader) 的 open-x4-sdk

## 项目状态

**当前版本: v0.1.0-dev**

这是一个个人项目，仍在积极开发中。已知问题：
- 首次打开图片需要转换为 BMP 缓存，稍有等待
- 大量图片（>200 张）时索引内存有限
- 灰度渲染需要 2 次屏幕刷新，这是 E-Ink 硬件限制
